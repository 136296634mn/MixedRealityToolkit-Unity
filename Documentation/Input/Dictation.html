    Dictation

    Dictation allows users to record audio clips and obtain a transcription of the audio. To use it make sure that a dication data provider is registered in the Input System Profile. Windows Dictation Input Provider is the dication service provided out of the box but alternative dictation systems can be created implementing IMixedRealityDictationSystem.
